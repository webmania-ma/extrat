Odoo 10: CRM Task Module

This module is used to create task form lead.


How to use

You can find create task button on the form of lead so if you want to create task form that lead you have to just click on that button.

Wizard for enter extra details releted to project task.

When you click on create task button wizard will be open where you have to enter details of project of that task,dedline of the task and assigned person details.

On lead form you can See Number of task created from that lead .


Here you can see the detailed information of the task created from the lead.
